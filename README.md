# covidsl.com

A COVID-19 Coronavirus tracker for Sri Lanka. Government agencies release detailed data.

### Requirements

 * NodeJS

### Build Instructions

Set up environment
```
npm install
```

Build once:
```
npm run build
```

Build continuously:
```
npm run watch
```

Start Server:
```
npm run start

# or run continuous build + server
npm run start-webpack
```

Build for production (minified):
```
npm run build-prod
```
